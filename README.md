# Welcome to Artifice

This is my final year Advanced Digital Media Project!
It is a virtual gallery inspired by exploration and play. It features a central gallery in an open environment with a cityscape sky. Inside it holds my personal favourite portraits and sculptures from a small collection of London exhibits as a regular gallery-goer. Furthermore, there are abstract geometric shapes in the open environment around the gallery - in which they are positioned for users to discover, encourage curiosity, and emphasises the digital aspect of the virtual space. 

## Enjoy!
